- username: superuser

## Project Setup

### Install

```bash
$ npm install
```

### Development

```bash
$ npm run dev
($ electron-vite dev --watch)
```

### Build

```bash
# For windows
$ npm run build:win

# For macOS
$ npm run build:mac

# For Linux
$ npm run build:linux
```
